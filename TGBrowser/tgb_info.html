Version History

warning_amber Note: Some older releases might not be listed here.
Stable Channel
v26.01.07d
From "Beta 5" to "Stable 5," you can now experience TGMod v3-secure in this release, as well as TGPanel animations.
v25.12.19b-r2
From "Beta 3" to "Stable 3," you can now experience TGMod v1 in this release, as well minor tweaks in the code.
v25.12.19a-r2
From "Beta 2" to "Stable 2", applied critical switch to Bing Search.
v25.12.18c-r2
Moved from VanillaUI to Material Design 2, improved TGPanel, homepage introduced, now defaults to Google Search.
v25.12.18b-r7
Minor adjustments, moved to Google Search.
v25.12.18a-r1
Added bookmark feature, introduced Bing Search capability.
v2.4.5alpha
Link only access, no google search, default browser theme (VanillaUI, stock appearance).

Beta / Experimental Channel
v26.01.22a
Beta 6:Features a new loading animation, new animations to TGBrowser's UI, and the addition of Material You (Material Design 3)
v26.01.07d
Minor adjustments to TGRitual theme modes and Dark Mode. New animation for TGPanel, smaller with rounded corners. Addressed TGRitual theme picker background.
v26.01.07c
Added a security system for TGMod, now checks for TGUID on all TGplugs loaded.
v26.01.07b
Updated TGMod system to load TGPlugs by folder name typing.
v26.01.07a
Beta 5: Features TGChroma, TGRitual, new "TGHome" system, dynamic animations on TGPanel. Refined TGMod now includes "TGMoLink" API for mod communication and enhanced loading.
v25.12.20b-br1
Removal of "Go Back" and "Go Forward" navigation buttons for space saving.
v25.12.20a-br2
Adjusted TGPanel's UI - from white to cream color.
v25.12.20a-br1
Beta 4: Added TGPanel UI transparency, minimal tweaks and adjustments.
v25.12.19a/b-br2
Applied the critical Bing Search switch.
v25.12.19b-br1
Introducing TGMod (Modding Interface) - loads JS or CSS files (tg_skin.css and tg_spirit.js format only).
v25.12.19a-br2
Removed "Masked Agent" (buggy), added "About Browser" clickable button (opens a local file).
v25.12.19a-br1
Introduced "Data Barrier (Cookie Blocker)" and "Masked Agent" (user-agent spoofing).
