About TGBrowser

What is TGBrowser?

TGBrowser is a lightweight, iframe-based web browser built entirely with HTML, CSS, and JavaScript. It's designed as a modular, extensible browser that runs directly in your web browser, offering features like bookmark management, privacy controls, theming, and a unique modding system called TGMod.

Core Philosophy

build Modularity

Built with extension in mind via TGPlugs (CSS/JS mods)

security Privacy-First

Includes Data Barrier for cookie blocking

palette Customizable

Visual themes, modes, and home screen customization

bolt Lightweight

No installation required, runs in GitHub Pages

Key Features

  • check_circleTGHome: Customizable home screen using GIFs or Images (PNG Only)
  • check_circleTGChroma: Dynamic color system for primary/accent colors
  • check_circleTGRitual: Special visual modes (Retro Sepia, Matrix Mode)
  • check_circleData Barrier: Sandbox-based cookie and storage blocking
  • check_circleBookmarks: Bookmark system with favicon support
  • check_circleSite History: Session history tracking (last 10 sites)
  • check_circleTGPlugs: A name for JS and CSS mod package
  • check_circleTGMod: The Freedom-for-All Mod system
  • check_circleTGMoLink API: TGMod's dedicated API for mod registration and communication

Technical Architecture

  • codeBase Engine: Iframe-based rendering with configurable sandbox permissions
  • storageStorage: Uses localStorage for bookmarks, settings, and homepage
  • searchSearch: Bing integration with search suggestions
  • shieldSecurity: Sandboxed iframes with optional cookie blocking
  • extensionExtensibility: Dynamic script/CSS loading for mods
