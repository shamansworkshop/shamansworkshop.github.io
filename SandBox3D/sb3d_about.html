About Sand Box 3D

What is Sand Box 3D?

Sand Box 3D is a fully interactive 3D physics sandbox built with Babylon.js and Cannon.js. It provides a creative playground where you can spawn objects, import custom models, apply physics forces, freeze objects in place, and cause spectacular explosions. Perfect for experimentation, stress testing, or just having fun with physics!

Tools & Settings

Game Settings NEW

A dedicated panel to optimize your experience. Click "Game Settings" to access:

  • speedFPS Limit: Cap framerate at 24, 30, 45, or 60 to save battery, or set to âˆž (Unlimited) for maximum smoothness.
  • visibilityView Distance: Adjust how far the camera renders (50m - 1500m). Lower this if you experience lag.
  • center_focus_strongFOV: Adjust Field of View (0.5 - 2.5) for a wider or zoomed-in perspective.
  • aspect_ratioResolution (Mobile): Scale resolution down to improve performance on older phones.

Standard Tools

remove_circle_outline None (Safe Mode): Disables all interaction.
visibility Possess: Take control of an object. WASD/D-Pad moves it.
pan_tool Force Push: Apply a directional impulse to objects.
ac_unit Freeze: Lock objects in place (Turn blue, Mass = 0).
delete Eraser: Delete objects (Ground/Ramp are protected).

Objects & Shapes

Custom 3D Models NEW

You are no longer limited to primitive shapes! Use the Upload 3D Model button to import your own files.

  • check_circleFormats: Supports .glb, .gltf, and .obj.
  • check_circlePhysics: Automatically generates a bounding-box hitbox for the model.
  • check_circleUsage: Once uploaded, select "Spawn Upload" in the Custom section to place copies.
