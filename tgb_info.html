TGBrowser Release Notes

About this Page

TGBrowser's dedicated page for version releases notes.

Releases

Some Releases might not show up here.

STABLE

v2.4.5alpha

Link only access, no google search, default browser theme (VanillaUI, stock appearance)

v25.12.18a-r1

Added bookmark feature, introduced Bing Search capability

v25.12.18b-r7

Minor adjustments, moved to Google Search

v25.12.18c-r2

Moved from VanillaUI to Material Design 2, improved TGPanel, homepage introduced, now defaults to Google Search

v25.12.19a-r2

From Beta 2 to Stable 2, applied critical switch to Bing Search.

v25.12.19b-r2

From "Beta 3" to "Stable 3,' you can now experience TGMod v1 in this release, as well minor tweaks in the code.

BETA

v25.12.19a-br1

Introduced "Data Barrier (Cookie Blocker)" toggleable feature, introduced "Masked Agent" (user-agent spoofing)

v25.12.19a-br2

Removed "Masked Agent" (buggy), added "About Browser" clickable button (opens a file via file directory, must have the file downloaded)

v25.12.19b-br1

Introducing TGMod (TGBrowser Modding Interface) - loads JS or CSS files next to the main .html file of the browser. (tg_skin.css and tg_spirit.js format only)

v25.12.19a-br2 & v25.12.19b-br2

Applied the critical Bing Search switch.

v25.12.20a-br1

Introducing "Beta 4," added TGPanel UI transparency, minimal tweaks and adjustments.

v25.12.20a-br2

Adjusted TGPanel's UI - from white to cream color.

v25.12.20b-br1

Removal of "Go Back" and "Go Forward" navigation buttons for space saving.

v26.01.07a

Introducing "Beta 5," this release features TGChroma, TGRitual, all new homepage system "TGHome," dynamic animations on TGPanel, as well as the refined TGMod, now has "TGMoLink" - an API for all TGPugs to use, also enhancement of how the system loads TGPlugs.

v26.01.07b

Updated TGMod system to load TGPlugs by folder name typing.

v26.01.07c

Added a security system for TGMod, now checks for TGUID on all TGplugs loaded.

v26.01.07d

Minor adjustments to TGRitual theme modes, as well as Dark Mode. New animation for TGPanel, slightly made it a bit smaller and rounded corners. Addressed TGRitual theme picker background.
