What is TGBrowser?

TGBrowser is a lightweight, iframe-based web browser built entirely with HTML, CSS, and JavaScript. It's designed as a modular, extensible browser that runs directly in your web browser, offering features like bookmark management, privacy controls, theming, and a unique modding system called TGMod.

Key Features

  • TGHome: Customizable home screen using GIFs or Images (PNG Only)
  • TGChroma: Dynamic color system for primary/accent colors
  • TGRitual: Special visual modes (Retro Sepia, Matrix Mode)
  • Data Barrier: Sandbox-based cookie and storage blocking
  • Bookmarks: Bookmark system with favicon support
  • Site History: Session history tracking (last 10 sites)
  • TGPlugs: A name for JS and CSS mod package
  • TGMod: The Freedom-for-All Mod system
  • TGMoLink API: TGMod's dedicated API that can do mod registration and give a communication system for TGPlugs

Technical Architecture

  • Base Engine: Iframe-based rendering with configurable sandbox permissions
  • Storage: Uses localStorage for bookmarks, settings, and homepage
  • Search: Bing integration with search suggestions
  • Security: Sandboxed iframes with optional cookie blocking
  • Extensibility: Dynamic script/CSS loading for mods
